# 🌐 Mon Portfolio Professionnel - Melissa Yahiatene

Bienvenue sur le dépôt de mon portfolio personnel. Ce projet présente mon parcours académique en **Informatique**, mes compétences techniques et mes réalisations.

## 🚀 Aperçu
Le site est déployé et consultable ici : https://melissa-yahiatene.github.io/mon-portfolio/

## 🛠️ Stack Technique
* **Frontend :** HTML5, CSS3 (Custom Properties, Grid & Flexbox).
* **Animations :** [AOS.js](https://michalsnik.github.io/aos/) (Animate On Scroll).
* **Iconographie :** Font Awesome 6.
* **Typographie :** Space Grotesk & Inter (Google Fonts).


## 📂 Structure du projet
* `/images` : Assets visuels et photos.
* `index.html` : Structure principale.
* `style.css` : Design et responsive.
* `script.js` : Logique d'interaction.
